# NightWatch Example

## 測試站台

- clone [todoMVC](https://github.com/bbandydd/React_Redux_Todolist) run at http://localhost:8888/

or

- run at https://bbandydd.github.io/React_Redux_Todolist/

## folders

- bin: chrome driver
- env: 環境變數設定
- test: 測試程式碼
- nightwatch.conf.js: nightwatch設定檔
- package.json: npm設定檔

## setup

```
npm install
```

## run test

```
npm run start
```
